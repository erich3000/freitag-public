Meine Projekte

2025-04-02 Entwurf

Neben meiner eigentlichen Arbeit treibe ich in meiner Freizeit eine Reihe von Projekten voran, die mir am Herzen liegen. Die meisten sind im Laufe der Jahre als Hobbyprojekte entstanden – manche klein und fokussiert, andere inzwischen zu einem richtigen Ökosystem herangewachsen.

Der Reiskoch

Der Reiskoch ist mein Koch-Blog mit Schwerpunkt auf Thai- und asiatischer Küche – 2011 gestartet und seitdem kontinuierlich gewachsen. Hier veröffentliche ich Rezepte, Zutatenerklärungen, Restaurantbewertungen und Kochbuch-Besprechungen. Technisch basiert der Blog auf Hugo und dem Khao Pad Hugo Theme , meinem eigenen Theme.

  • GitHub

Ahaan Thai

Ahaan Thai (อาหารไทย) ist mein Thai-Food-Lexikon – ein umfassendes Nachschlagewerk für thailändische Gerichte von A bis Z. Ursprünglich als Ergänzung zu Der Reiskoch entstanden, um Transkriptionen und Erklärungen für Thai-Gerichtenamen zu dokumentieren, wurde das Projekt irgendwann zu groß und auf eine eigene Website ausgelagert. Das Lexikon ist zweisprachig auf Deutsch und Englisch und verfügt über eine interaktive Suchfunktion.

  • GitHub

Mein FREItag

Mein FREItag – dieser Blog hier – ist mein persönliches Tagebuch nach dem „20-Prozent-Prinzip". Ein kreatives Ventil abseits des Alltags, wo ich über eigene Projekte, Technik, KI-Workflows und Alltägliches schreibe.

  • GitHub

Khao Pad Hugo Theme

Khao Pad (ข้าวผัด, „gebratener Reis") ist mein Hugo-Theme, das ich für Der Reiskoch und Ahaan Thai entwickelt habe. Es bettet das Khao-Malet-Designsystem und die Khao-UI-Web-Components in Hugo-Partials und Shortcodes ein. Das Theme bringt u. a. eine Thai-Food-Wörterbuch-Integration, Render-Hooks für Links sowie responsive Tabellen und Bildoptimierung mit.

  • GitHub
  • npm

Khao UI & Khao Malet

Khao UI (ข้าว, „Reis") ist meine UI-Komponentenbibliothek auf Basis von Svelte 5, die Web-Components bereitstellt. Khao Malet (เมล็ดข้าว, „Reiskorn") ist das zugehörige Designsystem mit CSS-Custom-Properties für Farben, Typografie und Abstände. Beide sind die technische Grundlage für Der Reiskoch und Ahaan Thai.

  • GitHub
  • Storybook

Ahaan Thai MCP Server

Der Ahaan Thai MCP Server ist meine Sammlung von MCP-Servern (Model Context Protocol) für die APIs von Ahaan Thai. Er bietet drei Server mit insgesamt 26 Tools – für das Wörterbuch, das Lexikon und die Kochbuch-Bibliothek – und ermöglicht KI-Agenten direkten Zugriff auf meine Thai-Food-Daten. Der Server läuft lokal (stdio) sowie remote unter https://mcp.ahaan-thai.de/mcp.

  • GitHub
  • Remote MCP

Ahaan Thai Claude Code Marketplace

Der Ahaan Thai Claude Code Marketplace ist mein Plugin-Marketplace für Claude Code mit Werkzeugen rund um Thai-Food und Rezepte. Er folgt dem Grundsatz, skriptbasierte Skills gegenüber MCP-Servern zu bevorzugen, da Skills nur bei explizitem Aufruf aktiv werden. Enthaltene Plugins: Thai-Food-Wörterbuch, Thai-Food-Lexikon, Thai-Kochbuch-Bibliothek sowie ein MCP-Fallback-Plugin.

  • GitHub

ji-agent-skills

ji-agent-skills ist meine Sammlung von Claude-Code-Plugins für KI-gestützte Workflows. Die Plugins erweitern Claude Code um wiederverwendbare Skills, Hooks und Agents. Enthaltene Plugins: agent-todos für die Verwaltung von Agent-Aufgaben, hugo-blog für Hugo-Workflows, obsidian für Vault- und Kanban-Integration, trello2obsidian für die Konvertierung von Trello-Exporten sowie skill-teaching zur Synchronisation von Skills mit anderen KI-Agenten.

Thai Food Dictionary MCP Server

2025-08-22 Drinnen

Vor ein paar Monaten hatte ich angefangen, mein Thai-Food-Lexikon ahaan-thai.de ins Englische zu übersetzen. Das Übersetzen der Markdown-Dateien, die Hugo dann als HTML rendert, habe ich anfangs noch mit deepl.com gemacht, bevor ich auf Claude Code umgestiegen bin. Das Übersetzen an sich hat damit auch sehr gut funktioniert, nur die unterschiedlichen Transkriptionen im Deutschen und im Englischen waren immer wieder ein Problem.

Da ich in dem Lexikon auch ein Thai-deutsches Wörterbuch für Begriffe rund ums Essen bereitstelle, kam mir die Idee, dieses per API (naja, wenn man ehrlich ist, ist es nicht mehr als ein JSON-Dump) verfügbar zu machen. Diese API habe ich wiederum per Model Context Protocol für Claude verfügbar gemacht. So kann der KI-Agent nun auf meine standardisierten Transkriptionen zugreifen und viele thailändische Wörter übersetzen. Der Clou dabei ist, dass Claude Code sowohl die Schnittstellen als auch den MCP selbst entwickelt hat.
